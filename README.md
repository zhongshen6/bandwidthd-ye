# bandwidthd-ye
Bandwidthd-ye 是一款专为网络管理员和开发者设计的轻量级、交互式流量分析工具。它能够将复杂的原始流量日志（如 Bandwidthd 导出的数据）转化为直观、美观的动态图表，帮助你快速识别网络中的“流量大户”与协议分布。
✨ 核心功能
📊 全维度数据展现：
折线趋势图：支持上传、下载及总流量的实时波动分析。
智能饼图：自动分析 IP 占比与协议分布，支持“点击联动过滤”。
全宽详细列表：平铺式设计，完整显示 IP 的 FTP、HTTP、SMTP、TCP、UDP、ICMP 详细流量明细。
🎯 极致的交互体验：
三态自适应布局：
全域模式：显示 11 列完整协议详情。
单项过滤：自动简化为 5 列核心指标，聚焦关键数据。
双重过滤：当同时筛选 IP 和协议时，自动隐藏详细栏，保持界面清爽。
页面级滚动：详细信息栏不再受限于内部滚动条，直接在页面平铺，带来丝滑的查阅体验。
响应式优化：针对窄屏设备优化列顺序，将“总流量”与“占比”置于首屏，减少横向滑动。
⚙️ 强大的自定义能力：
数据分桶 (Accuracy)：自定义各时间粒度下的采样点，在性能与精度间寻找完美平衡。
智能合并：自动将低流量 IP 合并为“其他”，并支持手动调节合并阈值。
上传/下载翻转：适配不同环境下的流量定义标准。
🔒 私有化与安全：
纯前端处理：数据解析与渲染全部在浏览器本地完成，数据无需上传，隐私零泄露。
自动加载：支持配置后端 JSON API 实现全自动化数据更新。
